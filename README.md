# 🐢 Turtle Dash

Игра на Python с использованием библиотеки turtle.

## Управление
- `W` — движение вверх
- `S` — движение вниз
- `Пробел` — старт / рестарт

## Механика
- Уворачивайся от красных черепах
- Собирай золотые монеты
- Рекорд сохраняется в файл `record.txt`

## Установка и запуск
```bash
git clone https://github.com/Jo1ann/Turtle-Dash
cd Turtle-dash
python "turtledashgame.py"
